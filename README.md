# Fragile - Unity 2D Runner Game

**Fragile** is a 2D endless runner game built in **Unity 2D**. The player controls a character running along a path, jumping over obstacles, collecting coins, and trying to reach a high score.  

---

## Gameplay Features

- **Character Movement**
  - Automatically runs forward
  - Jump over obstacles using **Spacebar** (keyboard) or touch (mobile)
  
- **Collectibles**
  - Coins scattered along the path
  - Each coin increases the **score**

- **Obstacles**
  - Avoid hitting obstacles or falling off the path
  - Hitting obstacles ends the game

- **Scoring**
  - Score increases over time and by collecting coins
  - Tracks **high score** for replay motivation

- **Endless Gameplay**
  - The game continues until the player collides with an obstacle

---

## Tech Stack

- **Engine:** Unity 2D (2019/2020/2021 or later)  
- **Language:** C#  
- **Assets:** 2D sprites for player, obstacles, coins, background  
- **Input Controls:** Keyboard (Space) or Mobile Touch
